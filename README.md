# Building Trust in the Age of AI

An interactive case study deck for **Identity for AI Agents** — Twilio's product for letting AI agents securely authenticate and act on behalf of a user, illustrated through a fictional trading app ("Owl Trade") connecting to ChatGPT.

**Live deck:** https://kellylimdesign.github.io/portfolio-2026/

## About

This is a single-file HTML/CSS/JS slide deck (no build step, no dependencies) with clickable, interactive prototypes built directly into the slides — including a live-feeling ChatGPT connection flow, a Twilio Console configuration wizard, and an SMS/iMessage approval mock — so the flows can be explored firsthand rather than just described.

## Structure

- `index.html` — the deck itself
- `mocks/` — product screenshots used throughout
- `demos/` — recorded product demo clips
- `assets/` — device-frame images used in mocks
